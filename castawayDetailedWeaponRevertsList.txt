Detailed Weapon Reverts Information

These are all the weapon reverts present on the revert servers. This list can be printed in console in-game using the !reverts command. 

Airblast 
	Airblast version currently used in this server is the present airblast version (post Jungle Inferno).

All Demoman's Swords
Reverted to pre-toughbreak, no holster and deploy penalty
	Swords now deploy and holster 75% faster again - the same as stock Demoman melee speed.

	User does not need to wait for 1 second for full damage and accuracy to ramp up any more.
